Create `artifacts/riskmind-app/src/components/dashboard/risk-heatmap-chart.tsx` with the following implementation:

**Props interface:**
```typescript
interface RiskHeatmapChartProps {
  cells: Array<{
    likelihood: number;
    impact: number;
    risks: Array<{ id: string; title: string; status: string; category: string }>;
  }>;
  onCellClick?: (likelihood: number, impact: number) => void;
  selectedCell?: { likelihood: number; impact: number } | null;
}
```

**Component implementation details:**

1. **Import `ReactECharts` from `echarts-for-react`** and `echarts` core types. Use `import ReactECharts from 'echarts-for-react'` (default export).

2. **CSS variable extraction function** — `getThemeColors()`:
   - Call `getComputedStyle(document.documentElement)` to read CSS variables
   - Extract `--severity-critical`, `--severity-high`, `--severity-medium`, `--severity-low` values
   - Extract `--muted`, `--card`, `--foreground`, `--muted-foreground` for background/text
   - Convert HSL strings (e.g. `"0 84% 60%"`) to `hsl(0, 84%, 60%)` format for ECharts
   - Store in a `useMemo` or recalculate on theme change

3. **Theme change detection** — Use a `MutationObserver` on `document.documentElement` watching for `class` attribute changes (the `.dark` class toggles). When detected, recalculate colors and force re-render via a state counter. Clean up observer in useEffect return.

4. **Data transformation** — `useMemo` that converts `cells` prop to ECharts heatmap data:
   - Build a 5x5 data array: `[impactIndex, likelihoodIndex, count]` where indices are 0-4
   - For each cell `(l, i)`: find matching cell from props, get `risks.length` as count
   - Fill missing cells with count 0

5. **Severity color function** — `getSeverityColor(likelihoodIndex: number, impactIndex: number)`:
   - Convert 0-based indices back to 1-based: `likelihood = likelihoodIndex + 1`, `impact = impactIndex + 1`
   - Compute `score = likelihood * impact`
   - Return: `score >= 15` -> critical color, `score >= 10` -> high color, `score >= 5` -> medium color, else low color
   - This is the per-cell `itemStyle.color` — NOT visualMap

6. **ECharts option object:**
   ```typescript
   const option: EChartsOption = {
     tooltip: {
       position: 'top',
       formatter: (params) => {
         // params.value = [impactIdx, likelihoodIdx, count]
         const [iIdx, lIdx, count] = params.value;
         const likelihood = lIdx + 1;
         const impact = iIdx + 1;
         const score = likelihood * impact;
         const severityLabel = score >= 15 ? 'Critical' : score >= 10 ? 'High' : score >= 5 ? 'Medium' : 'Low';
         const likelihoodLabels = ['Rare', 'Unlikely', 'Possible', 'Likely', 'Almost Certain'];
         const impactLabels = ['Negligible', 'Minor', 'Moderate', 'Major', 'Catastrophic'];
         return `<strong>${severityLabel}</strong> (Score: ${score})<br/>` +
                `Likelihood: ${likelihoodLabels[lIdx]}<br/>` +
                `Impact: ${impactLabels[iIdx]}<br/>` +
                `Risks: <strong>${count}</strong>`;
       }
     },
     grid: { top: 10, right: 10, bottom: 60, left: 80, containLabel: false },
     xAxis: {
       type: 'category',
       data: ['Negligible', 'Minor', 'Moderate', 'Major', 'Catastrophic'],
       name: 'Impact',
       nameLocation: 'middle',
       nameGap: 40,
       nameTextStyle: { fontWeight: 'bold', fontSize: 13, color: mutedForegroundColor },
       splitArea: { show: false },
       axisLabel: { color: mutedForegroundColor, fontSize: 11 },
       axisTick: { show: false },
       axisLine: { show: false },
     },
     yAxis: {
       type: 'category',
       data: ['Rare', 'Unlikely', 'Possible', 'Likely', 'Almost Certain'],
       name: 'Likelihood',
       nameLocation: 'middle',
       nameGap: 60,
       nameTextStyle: { fontWeight: 'bold', fontSize: 13, color: mutedForegroundColor },
       splitArea: { show: false },
       axisLabel: { color: mutedForegroundColor, fontSize: 11 },
       axisTick: { show: false },
       axisLine: { show: false },
     },
     series: [{
       type: 'heatmap',
       data: heatmapData,
       label: {
         show: true,
         formatter: (params) => {
           const count = params.value[2];
           return count > 0 ? String(count) : '';
         },
         fontSize: 14,
         fontWeight: 'bold',
         color: foregroundColor,
       },
       emphasis: {
         itemStyle: {
           shadowBlur: 12,
           shadowColor: 'rgba(0, 0, 0, 0.3)',
         },
       },
       itemStyle: {
         borderRadius: 4,
         borderColor: 'transparent',
         borderWidth: 2,
         color: (params) => {
           const [iIdx, lIdx, count] = params.value as [number, number, number];
           if (count === 0) return mutedColor;
           return getSeverityColor(lIdx, iIdx);
         },
       },
     }],
   };
   ```

   **CRITICAL:** Do NOT use `visualMap`. The severity color is determined by cell POSITION (L*I score), not by the count value. The `itemStyle.color` callback in the series handles this.

7. **Selected cell highlight** — When `selectedCell` prop is set, use ECharts `dispatchAction` via the chart ref to highlight the corresponding data point:
   ```typescript
   const echartsRef = useRef<ReactECharts>(null);
   useEffect(() => {
     const instance = echartsRef.current?.getEchartsInstance();
     if (!instance) return;
     if (selectedCell) {
       instance.dispatchAction({
         type: 'highlight',
         seriesIndex: 0,
         dataIndex: (selectedCell.likelihood - 1) * 5 + (selectedCell.impact - 1),
       });
     }
   }, [selectedCell]);
   ```

8. **Click handler** — Register `onEvents` prop on ReactECharts:
   ```typescript
   const onEvents = {
     click: (params: any) => {
       if (params.componentType === 'series') {
         const [iIdx, lIdx] = params.value;
         onCellClick?.(lIdx + 1, iIdx + 1);
       }
     },
   };
   ```

9. **Responsive sizing** — Use `style={{ height: '100%', minHeight: 500 }}` on the ReactECharts component. The parent container controls width. Pass `opts={{ renderer: 'canvas' }}` for performance.
