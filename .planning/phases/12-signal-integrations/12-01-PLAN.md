1. Create `src/lib/signal-feed-poller.ts`:
   - Import `cron from "node-cron"` (v4 default import).
   - Import `db, integrationConfigsTable, signalsTable` from `@workspace/db`.
   - Import `decrypt` from `./encryption`, `enqueueJob` from `./job-queue`, `adapters` from `../adapters/types`.

   - Define default schedules per D-04:
     ```typescript
     const DEFAULT_SCHEDULES: Record<string, string> = {
       nvd: "0 * * * *",        // hourly
       shodan: "0 0 * * *",     // daily
       sentinel: "*/15 * * * *", // every 15 min
       misp: "0 * * * *",       // hourly
       email: "*/5 * * * *",    // every 5 min (fallback for IDLE)
     };
     ```

   - `export async function startSignalFeedPoller()`:
     For each sourceType in DEFAULT_SCHEDULES, schedule a cron job that calls `pollSourceForAllTenants(sourceType)`.
     Log `[SignalPoller] Scheduler started for: nvd, shodan, sentinel, misp, email`.

   - `export async function pollSourceForAllTenants(sourceType: string)`:
     Query `integrationConfigsTable` where `sourceType` matches AND `isActive = true`.
     For each config: call `pollSingleConfig(config)` in try/catch.
     On error: update `lastError` on the config row. Log error but do NOT throw (other tenants must continue).

   - `async function pollSingleConfig(config: IntegrationConfig)`:
     Decrypt `config.encryptedConfig` via `decrypt()` + `JSON.parse()`.
     Look up adapter from `adapters[config.sourceType]`. If not registered, log warning and return.
     Compute `since` = `config.lastPolledAt ?? new Date(Date.now() - 24 * 60 * 60 * 1000)`.
     Call `adapter.poll(decryptedConfig, since)`.
     For each RawSignal returned: insert into `signalsTable` with `onConflictDoNothing()`. If inserted (returned row), enqueue `ai-triage` job.
     Update `lastPolledAt = new Date()` and `lastError = null` on the config row.

2. Wire into `src/index.ts`:
   - Import `startSignalFeedPoller` from `./lib/signal-feed-poller`.
   - Call `startSignalFeedPoller();` after `startAgentScheduler();` in the `start()` function.
