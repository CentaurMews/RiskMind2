       interface RiskHeatmapChartProps {

       - Result format: `${count}${arrow}${badge}` e.g. "4 ↑ !"
       - Reduce `fontSize` from 14 to 11 to avoid overflow in resized heatmap (per Research pitfall 1)
       - Add `label.overflow: "truncate"` to prevent cell overflow

    3. Enrich the tooltip formatter (D-07):
       - Compute total risks across all cells
       - Add "X% of total" line: `const pct = total > 0 ? Math.round((count / total) * 100) : 0;`
       - Add delta line: if cellDeltas exists, show "+2 since last period" or "-1 since last period" or "No change"
       - Format: `<strong>${severityLabel}</strong> (Score: ${score})<br/>Likelihood: ${label}<br/>Impact: ${label}<br/>Risks: <strong>${count}</strong> (${pct}% of total)<br/>${deltaText}`

    4. Add WCAG AA aria label support (D-29):
       - Add `aria-label` prop to the ReactECharts wrapper div
       - Construct screen reader text: "Risk heatmap with {totalRisks} risks across 25 cells"

    5. Preserve ALL existing behavior: severity colors via itemStyle.color callback, MutationObserver dark mode, emphasis styles, click events, selectedCell highlight.
