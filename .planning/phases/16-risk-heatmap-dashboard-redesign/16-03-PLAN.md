<!-- Plan 01 API endpoints -->
```
GET /v1/risks/dashboard?category=optional
  -> { postureScore, aboveAppetiteCount, aboveAppetiteDelta, cellDeltas, aboveAppetiteCells, domainSummaries, cells, appetiteConfigs, collecting? }

GET /v1/risks/snapshots?range=6M
  -> { snapshots: Array<{ date, compositeScore, aboveAppetiteCount, totalRisks }> }

GET /v1/risks/appetite
  -> { configs: Array<{ category, threshold }> }

PUT /v1/risks/appetite/:category  (admin only)
  body: { threshold: number }
  -> updated config
```
