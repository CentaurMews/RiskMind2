**Part A — Mobile heatmap fallback (artifacts/riskmind-app/src/components/dashboard/heatmap-grid.tsx):**

The component needs to render a severity summary list on mobile instead of the 5x5 grid. Use Tailwind's responsive `hidden`/`block` classes — no JS resize detection needed.

1. Compute severity buckets from `cells` prop at the top of the component:
```typescript
// Compute severity summary from cells data
const severitySummary = (() => {
  const counts = { critical: 0, high: 0, medium: 0, low: 0 };
  cells.forEach(cell => {
    const score = cell.likelihood * cell.impact;
    const count = cell.risks?.length || 0;
    if (score >= 15) counts.critical += count;
    else if (score >= 10) counts.high += count;
    else if (score >= 5) counts.medium += count;
    else counts.low += count;
  });
  return counts;
})();
```

2. Return both the grid and the mobile list, using responsive display classes:
```tsx
return (
```
