      4. Parse responses JSONB as AssessmentResponses
      5. Call `computeScore(template, responses)` from assessment-engine.ts

      Body: `{ questionId: string, sectionResponses: Record<string, QuestionResponse> }`.
      1. Set SSE headers (Content-Type: text/event-stream, Cache-Control: no-cache, Connection: keep-alive, X-Accel-Buffering: no)
      2. Call res.flushHeaders()
      3. Build prompt: "You are evaluating assessment responses. The user just answered question '{questionText}' with '{answer}'. Based on this response and the context of the assessment, determine if a follow-up question would help clarify or explore the answer further. If yes, respond with a JSON object: { text, type, weight, required }. If no follow-up needed, respond with null."
      4. Stream via `streamComplete(tenantId, { messages }, "assessment")`
      5. On done: parse response. If valid JSON question, send `data: { type: "done", followUp: parsedQuestion }`. If null/unparseable, send `data: { type: "done", followUp: null }`.

    });
