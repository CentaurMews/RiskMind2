The resulting block must be:
```typescript
res.setHeader("Content-Type", "text/event-stream");
res.setHeader("Cache-Control", "no-cache");
```

Do NOT change any other line in this file. This is a single-line insertion.

WHY: Without `res.flushHeaders()`, Express/Node.js buffers the 200 status and headers in memory until the first `res.write()` call. Cloudflare's proxy, seeing no data on the connection, may buffer the entire response until close — destroying SSE token-by-token semantics. `res.flushHeaders()` forces the HTTP response line and headers to be flushed immediately over TCP.
